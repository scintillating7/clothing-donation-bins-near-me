Clothing Bins Near Me

How this works:

If you agree to let this app know where you are, it will tell you where some clothing donation boxes are. If you don't agree, it will default to Chicago, IL, USA, because that's where this website was born.

Coming features: I'm gonna start with adding some markers manually and deciding how each marker will have metadata. I'm thinking something like if you click on it, you see date last verified and perhaps the name of the company on it or whatever. Then we need the ability to add markers or to say "not there."

Styled by Bootstrap, powered by Google API, coded with love.
